# 💕 纪念日提醒小程序

> 记录每一个特别的日子，珍藏每一份回忆

## 功能特点

- 📅 **纪念日管理** - 支持生日、纪念日、自定义类型
- 🔄 **循环周期** - 每天/每周/每月/每年提醒
- ⏰ **提前提醒** - 1/3/7天前温馨提醒
- 🎁 **心愿清单** - 记录想要的礼物，可指定发送对象
- 📝 **回忆记录** - 文字 + 图片，记录每个瞬间
- 📆 **时间轨迹** - 按年份查看历史纪念日和回忆

## 页面预览

| 首页 | 添加 | 详情 | 时间轨迹 |
|:---:|:---:|:---:|:---:|
| 纪念日列表 | 添加编辑 | 回忆+心愿 | 按年回顾 |

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/你的用户名/anniversary-miniprogram.git
cd anniversary-miniprogram
```

### 2. 导入小程序

1. 打开 **微信开发者工具**
2. 点击「导入项目」
3. 选择项目目录
4. 填写 AppID（没有可使用测试号）

### 3. 开通云开发

1. 点击开发者工具左上角「云开发」
2. 点击「开通」
3. 创建环境（记录环境 ID）

### 4. 配置数据库

在云开发控制台 → 数据库，创建以下集合：

- `anniversaries` - 纪念日
- `memories` - 回忆
- `wishes` - 心愿清单

### 5. 修改配置

打开 `app.js`，将 `'your-env-id'` 改为你的云开发环境 ID：

```javascript
wx.cloud.init({
  env: '你的环境ID',
  traceUser: true,
});
```

### 6. 运行

点击「编译」即可预览效果。

```
│   ├── timeline/         # 时间轨迹

└── utils/                # 工具函数
```
